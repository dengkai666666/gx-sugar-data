# 轮播图片放置说明

## 📁 文件夹位置
`agriculture-website/images/carousel/`

## 📸 图片命名规则

请按照以下命名放置5张图片到此文件夹：

1. **slide-1.jpg** - 广西甘蔗种植基地
   - 建议内容：大片甘蔗田、种植场景
   - 尺寸建议：1920x1080 或 1600x900 (16:9比例)

2. **slide-2.jpg** - 现代化制糖生产线
   - 建议内容：制糖工厂、生产车间、制糖设备
   - 尺寸建议：1920x1080 或 1600x900 (16:9比例)

3. **slide-3.jpg** - 甘蔗机械化收割
   - 建议内容：甘蔗收割机作业、机械化收获场景
   - 尺寸建议：1920x1080 或 1600x900 (16:9比例)

4. **slide-4.jpg** - 糖业科研创新
   - 建议内容：实验室、科研人员、甘蔗品种研究
   - 尺寸建议：1920x1080 或 1600x900 (16:9比例)

5. **slide-5.jpg** - 糖业产业园区
   - 建议内容：产业园区全景、现代化厂房、仓储物流
   - 尺寸建议：1920x1080 或 1600x900 (16:9比例)

## 📐 图片要求

### 尺寸
- **推荐尺寸**: 1920x1080 (Full HD) 或 1600x900
- **宽高比**: 16:9
- **最小尺寸**: 1280x720
- **最大文件大小**: 每张不超过2MB

### 格式
- **支持格式**: JPG/JPEG (推荐), PNG
- **不支持**: GIF动图、WebP

### 质量
- 清晰度高，无水印或仅有不明显水印
- 色彩饱和度适中，符合行业特点
- 避免过度曝光或过暗

## 🎨 展示效果

- 轮播图高度：500px（自动按比例缩放）
- 自动播放：是（每5秒切换一次）
- 过渡效果：淡入淡出
- 底部指示器：5个圆点，显示当前位置
- 左右切换按钮：鼠标悬停时显示
- 图片说明：底部半透明黑色背景，包含标题和描述文字

## 📝 注意事项

1. **所有5张图片都必须放置**，缺少任何一张都会导致轮播显示错误
2. **文件名必须完全一致**（包括大小写和扩展名）
3. 如果某张图片暂时没有，可以使用临时占位图
4. 图片内容应真实反映广西糖业实际情况，避免使用不相关的图片
5. 建议图片来源于官方渠道或授权使用，避免版权纠纷

## 🔄 更新图片

如需更换图片，直接替换对应文件名的图片即可，无需修改代码。

如需修改图片说明文字，请编辑 `index.html` 文件中的 `carousel-caption` 部分。
